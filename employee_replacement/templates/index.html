Welcome to TalentSaver

This Flask web application is designed to manage user authentication and facilitate employee data processing through machine learning techniques. It features user registration and login functionalities secured by hashed passwords. Once logged in, users can upload employee data in CSV format, which is processed to predict employee retention using a RandomForestClassifier model. The application also supports finding suitable replacement candidates for employees who leave the organization by leveraging the NearestNeighbors algorithm. The code ensures data integrity by using SQLAlchemy for database interactions, with a user model defined for managing user credentials and sessions. The application includes several routes to render different HTML templates for user interactions, upload files, and display results.
